# Headless Platform Api Server

- Headless Platform 은 인프라 없이 인증을 빠르고 쉽게 할 수 있는 서비스 사이드 프로젝트 입니다.

## Service Spec
- Java11 (Amazon Corretto 11)
- SpringBoot 2.7.11
- Spring Data JPA
- QueryDSL
- SwaggerUi
- MySQL 8

## 배포 환경
- AWS EC2
- Docker
- Apache Tomcat
- GitHub Action

## 제작도구
- Intellij
- PostMan
- Notion

## 사용 기능
1. 회원 관리
2. 앱 관리
3. 결제 관리
4. 사용량 관리
5. 인증 기능 관리
6. 알림톡 관리
